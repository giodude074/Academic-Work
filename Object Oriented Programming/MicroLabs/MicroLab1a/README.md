### Specifications:

```
Create a Main class with a main method. Inside the class, initialize an integer array with a length of 5 and initialize an int size with a value of 0.

Create an addPosInteger(int num) that  adds an integer at index size. After adding the number to the array, increment size.
Create a Scanner. Using a while loop, ask the user to either enter a positive integer to be added to the array, or -1 to quit.  If they do enter -1, exit the program,  otherwise, add the integer input into the int array using the addPosInteger(int num).
```
