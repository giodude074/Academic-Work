### Specifications:
Problem Description: Word guessing game.

```
Write a GUI program that lets user play guessing word game.

1. The user guesses a word by entering one letter at a time.

2. If the user misses seven times, the game is over and you should notify user.

3. The missed letters should be displayed so that user will not try incorrect letters again.

4. Once a word is finished, the user can press the Enter key to continue to guess another word.
```
